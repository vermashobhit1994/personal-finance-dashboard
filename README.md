# personal-finance-dashboard
A dashboard for managing my personal finances (savings and expenses)

### problem statement
user isn't able to manage finance

### Software Solution
create dashboard to manage finances

### user stories
1. view total balance in account
2. view total expenses done
3. view total savings 
4. expense vs income graph for all months from Jan to Dec
5. view balance, expenses, savings, transaction history amount in different currencies of various countries 
6. Savings plan
   - investment goal
   - emergency fund   
7. transaction history
8. recurrring bills for each month
9. categorywise expenses
categories for expenses and % of money spent on each of them. 
10. monthly expenses
11. budget planner
    - limit the expenses on different categories 
     
12. cashflow for different months
13. income sources 
14. wallet
